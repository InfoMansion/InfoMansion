== 소개

SSAFY 7기 공통6반 A607팀

Info`Mansion REST API 명세서입니다.

[[commont]]
== 공통사항

=== Header

|===
| name | 설명

| `Authorization`
| API를 사용하기 위한 인증 키
|===

=== 공통 Response Body

|===
| field | 설명

| `code`
| 응답 코드

| `message`
| 예외사항 내용 등 응답 메세지

| `data`
| 실제 반환될 데이터
|===
